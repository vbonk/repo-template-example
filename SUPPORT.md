# Support

Thank you for using this project! Here is how to get help.

## Getting Help

- **README FAQ** — Check the [README](README.md) for common questions and usage examples.
- **Existing Issues** — Search [open issues](../../issues) to see if your question has been answered.
- **Discussions** — If enabled, use [Discussions](../../discussions) for general questions and ideas.

## Reporting Bugs

1. Search [existing issues](../../issues?q=is%3Aissue+label%3Abug) to avoid duplicates.
2. Open a [new bug report](../../issues/new?template=bug_report.yml) with:
   - Steps to reproduce
   - Expected vs. actual behavior
   - Environment details (OS, language version, etc.)

## Requesting Features

Open a [feature request](../../issues/new?template=feature_request.yml) describing:
- The problem you are trying to solve
- Your proposed solution
- Any alternatives you have considered

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for development setup, workflow, and guidelines.

## Security Issues

**Do NOT open a public issue for security vulnerabilities.**
See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
